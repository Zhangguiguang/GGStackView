## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Installation

Copy the file 

[GGStackView.h](https://github.com/Zhangguiguang/GGStackView/blob/main/GGStackView/Classes/GGStackView.h)  
[GGStackView.m](https://github.com/Zhangguiguang/GGStackView/blob/main/GGStackView/Classes/GGStackView.m)

OR

[UIStackView+GGCustomSpacing.h](https://github.com/Zhangguiguang/GGStackView/blob/main/GGStackView/Classes/UIStackView%2BGGCustomSpacing.h)  
[UIStackView+GGCustomSpacing.m](https://github.com/Zhangguiguang/GGStackView/blob/main/GGStackView/Classes/UIStackView%2BGGCustomSpacing.m)

to Your Project.

## Usage

If you want set `UIStackView` custom spacing below iOS 11.0, you can use 

```
// UIStackView+GGCustomSpacing.h
- gg_SetCustomSpacing: afterView:

// GGStackView.h
- ggSetCustomSpacing: afterView:
```
